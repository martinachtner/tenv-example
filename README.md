# tenv-example

Example of how to use [tenv](https://github.com/tofuutils/tenv) in GitHub actions and locally

## Usage
Follow the instructions in the [tenv Getting Started](https://github.com/tofuutils/tenv?tab=readme-ov-file#getting-started) to install tenv locally.

Verify that tenv is installed correctly by running:

```bash
tenv --version
```
